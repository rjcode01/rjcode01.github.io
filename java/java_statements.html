Java Statements

In java language a statement is an executable combination of tokens ending with a semicolon mark. Statement are usually executed in secquence in the order in which they appear.

Classification of java statements are as follows

1. Expression Statement

2. Labelled Statement

3. Control Statement

4. Synchronization Statement

5. Guarding Statement

In java control statements further divided into three parts.

1. Selection Statement

2. Iteration Statement

3. Jump Statement

In java election statements further divided into three parts.

1. if

2. if-else

3. switch

In java iteration statements further divided into three parts.

1. while

2. do-while

3. for

In java jump statements further divided into three parts.

1. break

2. continue
